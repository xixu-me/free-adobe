### ✅ 系统要求

| 项目 | 要求 |

| 账户类型 | Windows 管理员账户 |

> ⚠️ **重要提醒**：GenP 工具可能会被杀毒软件误报为威胁，请在操作前暂时禁用所有安全软件。

## 🚀 激活步骤

请严格按照以下顺序执行，每个步骤都很重要。

### 步骤 1：安装 Creative Cloud

#### 1.1 运行安装程序

1. 找到下载的 `Creative_Cloud_Set-Up.exe` 文件
2. 右键点击文件，选择 **"以管理员身份运行"**
3. 等待安装程序启动

#### 1.2 登录账户

在安装过程中，系统会要求您登录 Adobe 账户：

- ✅ 请注册一个**新的免费账户**，或使用临时邮箱
- ❌ **切勿**使用任何已有订阅或曾开启过免费试用的旧账户

> 💡 **提示**：使用全新账户可以避免激活冲突问题。

#### 1.3 安装选项

- 如果在安装中出现 "Adobe Genuine Service (AGS)" 选项，请**不要**安装它

#### 1.4 配置 Creative Cloud

安装完成后，CC 会自动打开。请立即进行以下设置：

1. 点击菜单：`菜单 → 文件 → 首选项` (Menu → File → Preferences)
2. 在 **"常规" (General)** 标签页下，**取消勾选**以下三个选项：
   - ⬜ 登录时启动 Creative Cloud
   - ⬜ 关闭后在后台运行 Creative Cloud
   - ⬜ 自动保持 Creative Cloud 为最新状态

#### 1.5 退出 Creative Cloud

设置完成后，**彻底退出 CC**：

- 点击 `菜单 → 文件 → 退出 Creative Cloud`
- 确保 CC 已完全关闭（系统托盘中没有 CC 图标）

### 步骤 2：使用 GenP 激活 Creative Cloud

#### 2.1 启动 GenP

1. 进入您解压好的 GenP 文件夹（如 `D:\GenP`）
2. 找到 `GenP-v3.7.0.exe` 文件
3. 右键点击，选择 **"以管理员身份运行"**

#### 2.2 配置选项

1. 在 GenP 窗口中，切换到 **"Options"（选项）** 标签页
2. **勾选** ✅ "**Always search for ACC**"（始终搜索 ACC）选项
3. 点击 **"Save Options"** 保存设置

#### 2.3 搜索并激活

1. 返回 **"Main"（主页）** 标签页
2. 点击 **"Search"（搜索）** 按钮
3. 等待 GenP 查找文件（可能需要几分钟）
4. 在结果列表中，**勾选**顶部的三个 **Creative Cloud** 相关文件
5. 点击 **"Patch"（激活）** 按钮
6. 等待程序自动运行

#### 2.4 确认激活

当界面自动跳转到 **"Log"（日志）** 标签页时，表示 CC 激活已完成。

> ✅ **成功标志**：日志中显示激活成功的信息，没有错误提示。

---

### 步骤 3：安装 Adobe 应用程序

#### 3.1 配置 Creative Cloud 以管理员身份运行

1. 打开文件资源管理器，前往 CC 的安装目录：
   - 默认路径：`C:\Program Files\Adobe\Adobe Creative Cloud\ACC`
2. 找到 `Creative Cloud.exe` 文件
3. 右键点击 → `属性` → `兼容性` 标签页
4. **勾选** ✅ "**以管理员身份运行此程序**"
5. 点击 **"应用"** → **"确定"**

#### 3.2 安装应用程序

1. 启动 Creative Cloud
2. 在 **"Apps"（应用程序）** 页面，安装您需要的所有程序：
   - 例如：Photoshop、Premiere Pro、After Effects、Illustrator 等
3. 等待所有程序安装完成

> ⚠️ **重要警告**：安装完成后，**不要启动任何应用程序**！

#### 3.3 退出 Creative Cloud

所有应用安装完毕后，再次**彻底退出 CC**：

- 点击 `菜单 → 文件 → 退出 Creative Cloud`
- 确保 CC 已完全关闭

---

### 步骤 4：使用 GenP 激活 Adobe 应用程序

#### 4.1 启动 GenP

再次以**管理员身份**运行 `GenP-v3.7.0.exe`。

#### 4.2 配置选项

1. 转到 **"Options"（选项）** 标签页
2. **取消勾选** ⬜ "**Always search for ACC**" 选项
3. 点击 **"Save Options"** 保存设置

> 💡 **注意**：这次要取消勾选，与步骤 2 相反。

#### 4.3 搜索并激活应用程序

1. 返回 **"Main"（主页）** 标签页
2. 点击 **"Search"（搜索）** 按钮
3. GenP 会找到您刚刚安装的所有 Adobe 应用程序
4. 确保它们都**已被勾选**（默认会全选）
5. 点击 **"Patch"（激活）** 按钮
6. 等待激活过程完成（可能需要几分钟）

#### 4.4 确认激活

检查日志标签页，确认所有应用程序都已成功激活。

---

### 步骤 5：启动与使用

#### 🎉 激活完成

您现在可以正常启动和使用您的 Adobe 应用程序了。

#### ✅ 正确的启动方式

从以下位置启动应用程序：

- **开始菜单**
- **桌面快捷方式**
- **任务栏固定图标**

#### ❌ 错误的启动方式

**不要**通过 Creative Cloud 窗口中的 **"Open"（打开）** 按钮来启动程序！

> ⚠️ 通过 CC 窗口启动可能导致激活失效。

#### 🛡️ 首次运行提示

首次打开应用时，如果杀毒软件（或 Windows Defender）提示威胁：

1. 这是**误报**，可以放心忽略
2. 在杀毒软件的 **"隔离区"** 中**还原**该文件
3. 将其**添加为例外（白名单）**

## 🔄 后续维护：如何更新

### 📦 更新 Adobe 应用程序（如 Photoshop）

当您需要更新某个 Adobe 应用程序时：

1. **打开 Creative Cloud**，更新您想要的应用程序
2. 等待更新完成后，**彻底退出 CC**
3. 重复执行 **步骤 4**（激活应用程序）：
   - 确保 "Always search for ACC" **未勾选**
   - 搜索并激活更新后的应用程序

### 🔧 更新 Creative Cloud 本身

当 CC 提示自身需要更新时：

1. **打开 Creative Cloud**，更新 CC 应用程序
2. 等待更新完成后，**彻底退出 CC**
3. 重复执行 **步骤 2**（激活 CC）：
   - 确保 "Always search for ACC" **已勾选**
   - 搜索并激活 CC

### 📌 更新注意事项

| 更新对象 | Always search for ACC 选项 | 参考步骤 |
|----------|---------------------------|----------|
| Creative Cloud 本身 | ✅ 勾选 | 步骤 2 |
| Adobe 应用程序（如 PS、PR） | ⬜ 不勾选 | 步骤 4 |
